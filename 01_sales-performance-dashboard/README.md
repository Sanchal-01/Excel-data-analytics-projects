## About This Project

This is my first Excel-based data analytics project where I created an interactive **Sales Performance Dashboard** to analyze employee performance across different regions.

The main purpose of this project was to practice real-world Excel analytics concepts like Pivot Tables, slicers, charts, and data analysis ‚Äî not just formulas.

Using this dashboard, anyone can quickly understand:

- Top 5 performing employees  
- Bottom 5 performing employees  
- Total sales by employee  
- Target Hit Percentage (%)  
- Region-wise performance using slicers  

All visuals update dynamically when a region is selected.

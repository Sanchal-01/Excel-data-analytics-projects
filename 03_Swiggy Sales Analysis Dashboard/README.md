#  Swiggy Sales Analysis Dashboard (Excel)

An interactive Excel dashboard built to analyze large-scale food delivery data and uncover insights around sales performance, customer behavior, regional trends, and product demand.

This project transforms raw Swiggy order data into actionable business intelligence using Excel-based analytics and visualization techniques.

---

##  Project Purpose

The Swiggy Sales Analysis Dashboard is designed to explore ~197,000+ food delivery records and provide a clear view of platform performance across cities and states.  
The dashboard focuses on understanding revenue patterns, order trends, food preferences (Veg vs Non-Veg), and customer ratings to support data-driven decision making.

---

## Tech Stack

The dashboard was built using the following tools and techniques:

- 📊 Microsoft Excel – Core platform for analysis and dashboard creation  
- 📈 Pivot Tables & Pivot Charts – For summarizing and visualizing large datasets  
- 🧮 Excel Formulas – Data preparation and calculated metrics  
- 🎛 Interactive Slicers – Dynamic filtering by State, City, Month & Restaurant  
- 📋 KPI Cards – Total Sales, Orders, AOV & Average Rating  
- 📁 File Format – `.xlsx` (dashboard + analysis sheets)

---

## 📂 Data Source

Dataset consists of approximately **197,430 Swiggy food delivery records**, including:

- Order Date  
- State & City  
- Restaurant Name  
- Food Type (Veg / Non-Veg)  
- Order Value  
- Customer Ratings  

The data represents multi-city food delivery activity across India and was used to simulate a real-world analytics scenario.

---

## ✨ Features & Highlights

### 📍 Business Problem

Food delivery platforms generate massive amounts of transactional data, but extracting meaningful insights from raw tables is challenging.

Key questions addressed:

- Which cities and states drive the most revenue?  
- How do weekday vs weekend sales compare?  
- What is the performance of Veg vs Non-Veg items?  
- Which regions contribute the most orders?  
- How does sales vary month-over-month?

---

### 🎯 Goal of the Dashboard

To build an interactive Excel dashboard that:

- Provides a high-level performance overview using KPIs  
- Enables regional sales analysis by state and city  
- Tracks daily and monthly sales trends  
- Compares Veg vs Non-Veg revenue contribution  
- Supports business storytelling through visuals

## 📈 Business Impact & Insights

- Bengaluru emerges as the highest revenue-generating city.  
- Weekend sales consistently outperform weekdays.  
- Veg items contribute more to total revenue than Non-Veg.  
- Sales peak in later months, indicating seasonal demand trends.  
- A small number of cities account for a large portion of overall platform revenue.  
- Some restaurants operate across multiple states, while others are limited to a single region—impacting regional performance visibility.

These insights can help optimize marketing strategies, improve regional targeting, and support operational planning.

# Retail Sales Performance Dashboard

## About This Project

This is my second Excel data analytics project where I worked on a retail sales dataset and built an interactive dashboard after performing data cleaning and preparation.

The project focuses on understanding sales trends, profit distribution, customer behavior, and regional performance using Excel analytics tools.

---

## Project Workflow

1. Raw data inspection  
2. Data cleaning (dates, blanks, formatting, new columns)  
3. Feature engineering (Month, Year, Categories)  
4. Pivot Tables for analysis  
5. KPI creation (Sales, Profit, Quantity)  
6. Interactive dashboard using slicers and charts  
7. Key business insights

---

## Dashboard Features

- KPI Cards (Total Sales, Total Profit, Quantity Sold)
- Category-wise profit trend
- Sub-category sales distribution
- Top and Bottom performing states
- Monthly sales trend
- Customer count analysis
- Year, Month, and Category slicers

---

## Tools & Excel Features Used

- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques
- Date Functions
- Conditional Formatting
- KPI Cards
- Dashboard Design

---

## Key Insights

- Technology category generates highest profit over time.
- Phones and Chairs are top-selling sub-categories.
- California and New York are highest revenue states.
- Some states contribute very little to sales and need attention.
- Sales peak during September and November.
- Top customers contribute significantly to total profit.
